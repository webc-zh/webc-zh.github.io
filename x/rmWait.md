# 移除加载等待元素

无导出函数。导入时自动从 DOM 中移除 ID 为 `W` 的元素。

## 设计意图 & 为什么要放置等待动画于 `pug/_.pug`

在 `pug/_.pug` 中，定义了具有 ID `W` 的 `<style>` 标签，该样式利用 CSS `body:before` 伪类实现了一个无需 JS 依赖的纯 CSS 加载等待动画（旋转的圆环）。

这样做的目的是：

- **首屏秒开体验**：在浏览器解析 HTML 时即刻呈现加载状态，无需等待体积较大的 JS 脚本（或 Svelte 组件框架）下载、解析与执行完毕。
- **避免白屏**：防止网络较慢或 JS 文件加载时用户看到白屏，提供更好的用户视觉反馈。

## 哪儿被调用 / 移除时机

当主 JS 或组件脚本加载完毕并初始化好 DOM 后，会通过动态导入或模块导入执行 `x/rmWait.js`，瞬间移除 `<style id="W">`，从而销毁加载等待动画。

在 `page/entry/init.js` 结尾的地方导入，移除加载中的动画。
